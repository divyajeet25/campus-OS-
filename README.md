🌌 Campus-OS
Smart Academic Dashboard for Modern Students 🚀

Campus-OS is a futuristic student management ecosystem designed to transform academic life into a smooth, intelligent digital experience. It combines productivity tools, performance analytics, and elegant UI into one powerful platform.

Built with pure HTML, CSS & JavaScript, the project focuses on immersive animations, glassmorphism design, responsive layouts, and interactive effects to deliver a premium dashboard feel.

✨ Core Features

🔹 Live CGPA Calculator & Tracker
🔹 Smart Attendance Management System
🔹 Dynamic Performance Analytics Page
🔹 Focus Mode with Study Timer
🔹 Digital Library Management
🔹 Student Registration & Login UI
🔹 Animated Landing Page with Loader
🔹 Interactive Feature Cards
🔹 Liquid/Ripple Background Effects
🔹 Fully Responsive Design

🎨 UI/UX Experience

Modern Glassmorphism theme

Campus-OS is not just functional — it’s designed to feel powerful.
